Groupes d'utilisateurs

L'application peut utiliser les groupes du serveur Nextcloud pour limiter l'accès a une information à certain groupes d'utilisateurs seulement.

Principe d'Utilisation​

lors de l'ajout ou modification des menus, alertes , actualités et applications une entrée comme celle qui suit est présente.

Tous les groupes ajoutés dans cette case auront accès à l'information, contrairement aux membre de tous les autres groupes (y compris les utilisateurs sans groupes).

Si cette case est laissée vide , tout le monde peut voir l'information.
