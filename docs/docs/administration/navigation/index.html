Navigation

Présentation du menu de modification​

Un menu peut être de 3 type différent :

  • Section
  • Menu
  • Sous-menu

Chaque menu peut rediriger vers un lien ou fichier, à condition qu’il ne possède pas de menu « enfant ».

Seules les sections peuvent avoir des icones.

Le menu de modification se présente sous la forme de blocs, ici par exemple le bloc section ‘Vie des assemblées’ à pour menu « enfant » : « Règlement intérieur » et « Elus » …

Ajouter un menu​

Pour ajouter un menu il suffit de cliquer sur un bloc + au niveau et emplacement souhaité.

Cela ajoute un menu limité au groupe admin donc invisible pour les utilisateurs qui peut par la suite être modifier.

Modifier un menu​

Pour modifier un menu, il faut cliquer sur le bouton.

Dans ce menu vous pouvez modifier :

  • Titre du menu (attention si le titre est trop long le menu affichera des points de suspensions a la fin)
  • Lien URL OU redirection vers un fichier (une fois le fichier mis en ligne un lien local sera entrer dans le lien URL
  • Groupes utilisateurs (voir fonctionnement des groupes)

Supprimer un menu​

Il suffit de cliquer sur dans le bloc menu.

Réorganisation des menus​

L’ordre des menus peut être changer par un glisser-déposer en maintenant l’icône sur le menu souhaité.

Les menus ne peuvent être réorganiser que dans leur propre colonne, une section restera toujours une section, idem pour les menus et sous-menus.

Il est également possible de déplacer un menu de section et un sous-menu de menu, le menu déplacé sera positionné en dernière position nouvelle section / menu.
