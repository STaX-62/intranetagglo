Controllers

Fonctions des Controller de l'app et routes de l'API

Page​

PageController::index()​

  • description: permet l'affichage de l'application
  • url : /
  • Request Type : GET

PageController::getUserInfo()​

  • description : renvoie le nom, les groupes et la date de dernière connexion de l'utilisateur
  • url : /user
  • Request Type : GET

PageController::getLocation()​

  • description : renvoie un booléen après avoir vérifier si l'utilisateur était sur la plage ip de la CA2BM ou non
  • url : /location
  • Request Type : GET

PageController::searchGroups()​

  • description : renvoie les groupes de Nextcloud (max: 200)
  • url : /api/groups
  • Request Type : GET

Menus​

MenuController::index()​

  • description : renvoie tous les menus présents en BDD
  • url : /menus
  • Request Type : GET

MenuController::indexG()​

  • description : renvoie les menus présents en BDD trier en fonction des groupes de l'utilisateur
  • url : /menusG
  • Request Type : GET

MenuController::create()​

  • description : permet l'ajout d'un menu
  • url : /menus
  • Request Type : POST
  • Paramètres :
ParamètresTypeDescription
titleStringtitre du menu
iconStringicône du menu (voir icônes)
linkStringlien url du menu
groupsStringlimite de groupes du menus (voir groupes)
sectionidIntegerid de la section parente (ou 0 si pas un menu)
menuidIntegerid du menu parent (ou 0 si pas un sous-menu)
levelIntegerNiveau de l'actualité (0: Section, 1: Menu, 2: Sous-menu)

MenuController::update()​

  • description : permet la mise à jour d'un menu
  • url : /menus/{id}
  • Request Type : POST
  • Paramètres:
ParamètresTypeDescription
idIntegerid du menu
titleStringtitre du menu
iconStringicône du menu (voir icônes)
linkStringlien url du menu
groupsStringlimite de groupes du menus (voir groupes)

MenuController::changerOrder()​

  • description : permet de changer l'ordre des menus
  • url : /menus/order
  • Request Type : POST
  • Paramètres:
ParamètresTypeDescription
actualPositionStringposition actuelle du menu (voir position des Menus)
newPositionStringnouvelle position du menu (voir position des Menus)
sectionposInteger / nullid de la section parente (ou null si pas un menu)
menuposInteger / nullid du menu parent (ou null si pas un sous-menu)

MenuController::destroy()​

  • description : permet de supprimer un menu
  • url : /menus/{id}
  • Request Type : DELETE
  • Paramètre: id via url

News​

NewsController::index()​

  • description : renvoie les actualités en fonction des groupes de l'utilisateur
  • url : /news/{startid}
  • Request Type : POST
  • Paramètre: startid via url

NewsController::alerts()​

  • description : renvoie les alertes en fonction des groupes de l'utilisateur
  • url : /alerts
  • Request Type : POST
  • Paramètre : search

NewsController::create()​

  • description : permet de créer une actualité
  • url : /news
  • Request Type : POST
  • Paramètres:
ParamètresTypeDescription
titleStringtitre de l'actualité / alerte
subtitleStringsous titre de l'actualité (champ laissé vide sur une alerte)
textStringcontenu de l'actualité / alerte
categoryStringCatégorie de l'actualité (champ laissé vide sur une alerte)
groupsStringlimite de groupes de l'actualité / alerte (voir groupes)
linkStringlien de l'actualité (champ laissé vide sur une alerte)
expirationIntegerTimestamp en millisecondes (champ laissé à 0 pour une actualité)

NewsController::update()​

  • description : permet de mettre à jour une actualité
  • url : /news/update/{id}
  • Request Type : GET
  • Paramètres:
ParamètresTypeDescription
idIntegerid de l'actualité / alerte
titleStringtitre de l'actualité / alerte
subtitleStringsous titre de l'actualité (champ laissé vide sur une alerte)
textStringcontenu de l'actualité / alerte
categoryStringCatégorie de l'actualité (champ laissé vide sur une alerte)
groupsStringlimite de groupes de l'actualité / alerte (voir groupes)
linkStringlien de l'actualité (champ laissé vide sur une alerte)
expirationIntegerTimestamp en millisecondes (champ laissé à 0 pour une actualité)

NewsController::publication()​

  • description : permet de publier ou cacher une actualité, crée les notifications nextcloud
  • url : /news/pub/{id}
  • Request Type : POST
  • Paramètres:
ParamètresTypeDescription
idIntegerid de l'actualité / alerte
visibleBooléenvisiblité de l'actualité

NewsController::pinNews()​

  • description : permet d'épingler une actualité
  • url : /news/pin/{id}
  • Request Type : POST

NewsController::getCategory()​

  • description : permet de récupérer les catégories déjà créées
  • url : /news/category
  • Request Type : GET

NewsController::destroy()​

  • description : permet de supprimer une actualité / alerte
  • url : /news/{id}
  • Request Type : DELETE
  • Paramètre: id via url
